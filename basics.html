Intro to HTML

HTML stands for HyperText Markup Language. HTML provides the structure of our webpages by putting tags around text and images (otherwise known as marking up the text).

History of HTML

  • HTML 1.0 - developed in the early 1990s
  • HTML 2.0 - released in 1995
  • HTML 3.2 - released in 1997
  • HTML 4.01 - released in 1999
  • HTML 5 - released in 2014 - living standard/current version

Intro to CSS

CSS stands for Cascading Style Sheets. CSS can control the layout, colors, fonts - any visual element on the page can be controlled with CSS.

History of CSS

  • CSS1 - released in 1996
  • CSS2 - released in 1998
  • CSS3 - released in 2011 - living standard/current version

Basic Structure

    selector{
        property: value;
    }

The CSS groups of 3

3 ways to include CSS in our pages

  • External (seperate CSS file)
  • Embedded (place the CSS on a single HTML page)
  • Inline (add CSS to a particular HTML element)

CSS Selectors

  • HTML element (easiest, most global)
  • Custon class (we create a selector - add . at the beginning)
  • ID of an HTML element (add # at the beginning of the selector name)
